# Customer Funnel Optimization

## Business Analysis Capstone Project

### Project Overview

This project analyzes customer behavior across an ecommerce purchase funnel to identify opportunities for improving conversion performance, reducing cart abandonment, and increasing revenue.

The analysis was performed using:

- Microsoft Excel (EDA)
- MySQL (Business Analysis)
- Power BI (Dashboard Development)

---

## Business Problem

Ecommerce businesses attract a large number of visitors; however, many customers leave before completing a purchase. Understanding where customers drop off and what factors influence conversions is critical for improving business performance.

This project investigates:

- Customer funnel performance
- Cart abandonment behavior
- Marketing channel effectiveness
- Product category revenue contribution
- Customer engagement impact on purchases

## Key Findings

- Cart abandonment rate exceeded 65%.
- Customer engagement positively influenced conversion rates.
- Marketing channels showed varying conversion performance.
- Revenue generation was concentrated within specific product categories.
- Significant opportunity exists to improve funnel efficiency.
